# LectureForge AI Version 1

## Status
Working locally.

## Features
- YouTube URL input
- Backend transcript extraction
- Agent progress polling
- Study kit generation
- Timestamped outline
- Three-level summaries
- Flashcards
- Concept map
- Semantic search
- Transcript viewer

## Stack
- FastAPI backend
- React + Vite frontend
- Tailwind CSS
- React Flow
- OpenAI API
- youtube-transcript-api
- yt-dlp fallback

## Known limitations
- Some YouTube videos may block automated transcript or audio access
- In-memory job storage only
- Local development setup only
- No deployment yet
